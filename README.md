### Installation

This role requires at least Ansible `v1.6.3`. 

To install it, run:

    ansible-galaxy install AlbanAndrieu.css



### Role variables

List of default variables available in the inventory:

```yaml
    ---
    css_enabled: yes                       # Enable module
    css_editor_enabled: yes                # Enable editor
    ruby_version: 1.9.3
```
